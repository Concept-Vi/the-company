## Stress-test results (the questions this folder was meant to answer)
- **Is density coupled to ratio?** → **No — independent at the language level.** This is the *detailed/long* deck and it uses **3:2 (1893×1287)**, but the rules hold unchanged: side margins still **~12%** (measured 11.6%), zoning wash `#faf9f7` (≈ warm-ivory `#fdfcf7`), base `#fdfdfd`, gold ramp `#c4ab43`/`#c49649` (same hue family). The taller canvas was *chosen* for more content, but **% margins, type scale, zoning, gold are all ratio-invariant.** You could have a dense 16:9 or a sparse 3:2; ratio and density/LOD are **correlated by choice, not mechanically coupled.**
- **Does the ink-budget / whitespace rule survive high density?** → **Yes.** Higher detail is expressed as **more *sections* per slide** (multiple bronze sub-headers — e.g. "The Traditional Process" + "The Core Problem" on one slide; "Market Testing and Response" over a 2-panel block), **not** crammed text. Negative space stays generous. Density buys *structure*, not clutter.
- **Does the LOD ladder hold a third point?** → **Yes.** This deck sits **above** pitch-deck on detail: ~49pp, multi-section slides, fuller prose, more evidence. Ladder now has 3 real rungs: `deck1-2026` (terse/presented) < `pitch-deck` (standard/leave-behind) < `recent-pitches` (high-detail/long).
