Operator Controls

The operator-grade component set — Toast · Sheet · List · Menu · Checkbox/Radio · AppShell — plus the React wrappers over the CSS-only trio (Table · Search · Skeleton). Everything below is the cv-* classes the components render, generated from data. Tones ride the Badge vocabulary; selection is gold (the decision colour); row heights ride --row-h; the shell rides tokens/device.css chrome and the data-surface machinery.

Toast — the global notifier

TONES — THE BADGE VOCABULARY (gold · success · warning · error · comm) + ACTION SLOT

Sheet — slide-in panel (Modal's mobile-first sibling)

List — rows with leading · primary/secondary · trailing

DEFAULT DENSITY · .interactive SELECTION

data-density="compact" — SAME MARKUP, --row-h RECOMPUTES

Menu — action menu on the shared popover surface

Checkbox · Radio — gold = selection / decision

AppShell — nav rail (desktop) → bottom tab bar (mobile)

The CSS-only trio behind the React wrappers

TABLE — .cv-table--striped · __num · __feature

SEARCH — .cv-search
