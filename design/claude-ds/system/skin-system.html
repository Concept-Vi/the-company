The skin registry

One composition, every world

A skin is a coordinated point across the existing axes — ground, the default block surface, shadow character, thread language, state glow — scoped by data-skin and registered on the skin axis. The graph below is ONE spec: blocks as nodes, luminous threads as relations, states (selected · potential · stacked) as slots. Flip the skin and every role re-binds — the dark refractive glass world, the champagne-plaster stone world of the reference boards, fibrous parchment, machined carbon, and whatever is registered next — same tokens, same blocks, same resolutions.
