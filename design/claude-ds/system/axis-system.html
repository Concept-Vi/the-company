The Axis System

Every primary visual dimension is a first-class Axis in window.CV_AXES — a typed hierarchy of values. This page is a live projection of the registry (the interface reads the system, it is not a parallel list). Tokens are the value-units: each value shows its resolved token where it has one. Everything visual resolves its value through these axes; a component subscribes to a subset. Plan: analysis/AXIS-REFACTOR.md · model: system/glyphic-system.html §08.
