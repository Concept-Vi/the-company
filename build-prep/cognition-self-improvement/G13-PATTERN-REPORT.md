taught, re-taught, and (increasingly) followed. That is the right result for a mirror: it confirms
empirically, it does not fabricate novelty.

## Two BORDERLINE-NEW patterns — drafted for Tim's review, NOT minted

Per the no-fiction discipline these are *proposed*, since each is partially covered by existing notes:

**1. describe-vs-request discrimination** (`no-action-requested`, weight 13). A recurring exchange
shape: Tim thinks aloud / describes a problem / shares an observation — and the right move is
assessment, not action. The cost of misreading it as a request is unwanted work and churn.
*Draft rule: "When Tim is describing or musing, the deliverable is my read of it — act only when the
message asks for action. When ambiguous, give the assessment AND name what I would do next, without
doing it."* (Partially covered by the collaboration-frame note; this names the trigger.)

**2. build-and-show, don't spec** (inside `build-on-existing`'s examples). The correction "stop
writing specs for non-developers, start building and showing the interface in action" recurs in
different forms. *Draft rule: "When the choice is between writing a document about the thing and
building a showable slice of the thing, build and show — Tim evaluates running artifacts by sight,
not specs."* (Adjacent to render-for-cognition + tentative-artifact; this is the sharper operational
form.)

## What the clusters are USED for (the loop this report closes)

The history space is live and queryable — a fresh session can ask "what has Tim corrected about X"
and get the actual moments back ranked. This report is the REDUCE over that space; when future
minings shift the weights or surface a genuinely uncovered cluster, this file updates in place and
the new pattern arrives as a drafted rule for Tim's approve — the same propose→gate shape as
everything else.

Tail note: the long tail (700+ single-tag clusters) is mostly flow vocabulary, not failure signal;
the conservative threshold under-merges by design. A diverse-panel pass (GC7 machinery, once built)
could name the mid-tail better — logged, not blocking.
