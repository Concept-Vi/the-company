## Verification protocol (how a line earns ✅ — read before checking anything)
- **By USE, never code-reading.** A criterion is ✅ only when the real behaviour moves against real data/at scale where relevant, the output is inspected, AND an adversarial re-check (a pass that tries to DISPROVE "done") survives. Pattern-matching is not verification; code similarity proves structure exists, not that execution works. **A source-line that *states* a thing is true (a docstring, a comment, a shared call-site) is a code-read, NOT a by-use verification** — it earns 🟡, never ✅.
- **Falsify-first for every VERIFIED-BUG line.** Where a verified defect is named (the `structured_output` drop, the census error), the ✅ bar is: **write the failing test against UNMODIFIED code FIRST** (no fail = no bug = re-investigate), then it passes. No "fixed" without the red→green it proves. **Corollary (a non-bug must be PROVEN green):** where this doc claims a NON-bug (the `_RESOLVABLE` drift-guard, GR2), the falsify-first test must come up **GREEN on unmodified code** — that green is the proof there is no bug, and the test stays as a drift-guard.
- **FORM is verified by a SEPARATE design-critic + a fail-loud design-lint, on BOTH faces, rendered-for-cognition (Rule 9, native-mobile-always, render-for-cognition).** No FORM line is ✅ on the implementer's own grade. The FORM ✅ bar is FIVE clauses: (a) a **separate design-critic agent** passes it (the implementer cannot grade its own form); (b) the **design-lint passes** (off-token / bespoke-element → the build cannot be marked done) where machine-checkable; (c) **BOTH faces ship — native desktop AND native mobile** (the phone face is half of done); (d) the result is **surfaced for review** (a `decision.surfaced_for_review` event + a review inbox item — never a silent terminal); (e) **render-for-cognition (rule 13, confirmed operator rule)** — the surface renders so SHAPE carries the information (density / convergence / gaps / dead-ends are read at a glance, "his perception is the compute"), not a wall-of-text/flat-list that passes a generic critic while failing "shape carries the meaning."
