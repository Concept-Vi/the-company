Spacing throughout the interface is inconsistent and does not follow a system. Card padding varies between views. Gap between sidebar items is different from gap between board cards. The header-to-content spacing is different from content-to-footer spacing. Margin between stats numbers and labels is arbitrary. The token system defines spacing tokens (sys-space-spacing-xs through sys-space-spacing-xl) but layout uses hardcoded or default values. A consistent spacing scale must be applied to every element: cards, sections, navigation items, headers, content areas.
