The Timeline view renders as a flat vertical list of cards, identical in appearance to the Notice Board. There is no visual timeline — no time axis, no progression indicator, no visual encoding of when things happened relative to each other. A timeline should show time spatially: a vertical or horizontal axis with events placed along it, groupings by day/week, visual density showing activity periods, connectors between related events. Currently it is just another card list with "YESTERDAY 73 items" as a plain text divider. The word "Timeline" implies temporal visualization — the current view delivers none.
