Ask an advocate "what should we build next?" and the only answer it can return is "more of my era" — that is the advocate functioning exactly as designed, which is precisely why it is the wrong place to source cross-cutting DIRECTION. A direction sourced from a role-biased advisor has NO TERMINUS: each advocate is a gradient toward its own charge, so "what's next?" asked of one is a confirmation loop that can never resolve (reading the thermometer to decide the weather).

Treat "what should we do next?" answered by an advocate the same as channel-relayed-is-proposal: suspect until anchored to the named priority.
