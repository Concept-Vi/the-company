THE PIPELINE (Tim's light→big instinct + deterministic code bookends):

  (5) RECONCILE (RHM + Tim, in-channel) — flagged ambiguities -> a DECISION-CARD the RHM walks through WITH Tim -> back to the channels.
  (6) REGISTER (code) — vi-vision:// address + corpus + resolution + persist -> LIVE (keystone renders · DNA themes · company resolves).
