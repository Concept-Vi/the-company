Today an artefact is a static HTML page hosted in a sandboxed iframe. The capability generalises: an artefact can be a LIVE view that queries the engine — a run-monitor, a recall browser, a findings board are all just artefacts. So "host an artefact" quietly becomes "host any tool". The surface becomes where interactive views live, not just static pages. (Open Q: how a sandboxed artefact safely calls back to the engine.)
