Agent audit of 138 component_theme_specs found: color and border tokens are universal requirements, but layout spacing (gap, padding, margin, display, flex-direction, align-items) are NOT in the required_tokens for most components. Only 2 of 7 project space components require spacing. token_vocabulary HAS 56 spacing tokens (sys-space-gap-*, sys-space-inset-*, sys-space-spacing-*) but they're not listed as required in component_theme_specs. This means compose_block returns the tokens but components don't declare they need them. Class: component-specs-incomplete-for-layout — the tokens exist, the specs don't reference them. Fix: update component_theme_specs for all project space components to include layout tokens in required_tokens.
