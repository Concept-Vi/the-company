## 1 · Two independent axes (don't collapse them)

The real option space is a grid, not a line:

**Axis A — WHERE the vault sits:**
- **A1 · Point straight** at `channel-memory/noticeboard/` (the vault root *is* the live store).
- **A2 · Mirror** into a separate vault folder (a sync/projection of the store).

**Axis B — HOW MUCH we light up (graduated):**
- **L0** — Search + Properties only (zero shim).
- **L1** — Bases/Dataview "board views" (zero shim, reuse existing `.base` files).
- **L2** — Thin one-way enrichment: body-level wikilinks so graph/backlinks light up (read-mostly, runtime-safe).
- **L3** — Round-trip writeback: Obsidian edits flow back into the canonical store (the hard one).

Every cell A×B is a real configuration with a different cost. The honest default for "host not master" is **L0–L2, point-or-mirror**; L3 is where "host" risks quietly becoming "master."
