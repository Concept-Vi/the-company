The current chat is a placeholder textarea+send button. Tim confirmed it doesn't even work on his device and needs a total redesign. Design research filed at project://block-composition/keeper-agent/keeper-chat-total-redesign with 2026 AI chat patterns, mobile/desktop specifications, and implementation approach. Key elements: (1) Fix the send/receive pipeline so it actually works reliably. (2) Mobile: full-screen 85vh sheet with drag handle, Vi logo with animated states (breathe/pulse/glow), suggestion chips, tool activity indicators, voice input button. (3) Desktop: resizable side panel, slash commands, markdown preview, expandable tool call details. (4) Shared: token streaming with mid-stream markdown, conversation persistence, context awareness, actionable buttons in responses, connection status indicator. Implementation through composition system: keeper_chat component with template variants, chat-specific tokens, behavior bindings for send/minimize/voice actions.
