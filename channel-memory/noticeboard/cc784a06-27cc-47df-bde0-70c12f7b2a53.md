view_mode_registry defines icons for every view: Overview=layout-dashboard, Notice Board=clipboard-list, Board=columns, Timeline=clock, Graph=git-branch, Settings=settings. These are Lucide icon names. The tab bar renders plain text only. On mobile, icon-only tabs appear but without labels. Desktop should show icon+label tabs. The registry is the source of truth — the renderer should read view_mode_registry and render icons from it, not hardcode tab labels.
