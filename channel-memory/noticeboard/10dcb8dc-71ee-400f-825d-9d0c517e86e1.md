The Board view column headers (ACTIVE, OPEN, RESOLVED, CLOSED, IN PROGRESS) are plain text on colored backgrounds. The colors are basic (green, amber, teal, gray, blue) with no refinement. The column count number sits awkwardly. "right-click card to move" instruction text is visible — this is a developer hint, not user-facing design. Columns need: refined color tokens from the palette, proper typographic hierarchy, item count styled as a badge, drag-and-drop affordances instead of right-click instructions, smooth column entrance animations.
