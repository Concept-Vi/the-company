I’m gonna make an extra comment about iconography, I know you don’t really know this but in the DNA system, icons are a big deal. It goes to a lot of other things that I’ve said, about minimising text about minimising cognitive expense about engineering the experience to use The useless brain for them, iconography plays a big role in this. The V is a component, it is a special one because that symbol represents the identity of this system, but it’s just a symbol. It’s just an icon. An icon inside a shape. That same component could have any icon in it, all of the colours and the animations in the event actions, they can only be Setup to do anything just like the.  V is. So yeah, this comment on icons, that carries a lot more weight, and that fits into the design mentality.
