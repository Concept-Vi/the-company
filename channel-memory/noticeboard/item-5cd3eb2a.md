# Breakdown · The overlay / radial command menu (the V-symbol)

SPECIFIED: An overlay menu (like the company's "V" symbol that opens things) — maybe radial. Inside: switch element-mode / block-mode; submission controls; a count of comments made (tap one → it takes you to it on screen); a supervisor CHAT section; a section for responses / correspondences needing his eyes; whatever else is needed. Closed = non-distracting, full-screen content, selections still usable. Opens when he needs things.

IMPLIED: A persistent, unobtrusive, thumb-reachable entry point (a floating button) that expands to the session COMMAND CENTER: mode toggle · draft queue + submit-all · replies/inbox · supervisor chat · items-needing-attention (decisions/choices). Badges for pending count + unread replies. Tapping a queued comment navigates to its anchor. Must never block content interaction when closed. One-handed iOS ergonomics.

ADDITIONS: Quick-submit from the menu; jump-to-next-reply; a session indicator (active channel/members); settings (default selection level, haptics on/off); a "needs your eyes" priority lane; gentle animation; the V opens toward the thumb.
