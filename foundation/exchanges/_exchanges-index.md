# Foundational Exchanges — Archive

This folder holds the **verbatim primary sources** of the dense conversational exchanges that built the foundation of [[TIM|the Company]]. Each file holds one Tim message (verbatim, as dictated via TTS — `[bracketed]` text marks any readability correction) and the verbatim response, plus links to what that exchange produced.

These exist because **the conversation IS the construction** (see [[11-files-obsidian]]). The artefacts they produced (the synthesis layer in [[../system/README|`../system/`]], [[TIM]], and the memory entries) link back here; this index links forward to them. Bidirectional. The exchanges are the *source layer* (immutable, sequential, dated); the synthesis in `../system/` is the *doctrine layer* (timeless, integrated, no versioning).

> [!info] How to use this archive
> - Read an exchange to see *exactly* what Tim said and *exactly* how I responded — not a paraphrase.
> - Follow the **What this produced** links to see where each exchange landed in the foundation.
> - New dense exchanges get added here as they happen, numbered in sequence.
