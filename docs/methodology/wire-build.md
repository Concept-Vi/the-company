What it does each fire after step 7:

     - **CONFIRM/SURFACE/LOCKED** declared class (`posture(declared_class)!=AUTO`) ŌåÆ leave for the operator (surfaced; do NOT auto-dispatch). `dispatch_decision` also refuses these itself (no-bypass).

   - **┬¦W7 concurrency cap:** `drive_dispatchable` dispatches at most `implement.CONCURRENCY_CAP` (env `COMPANY_WIRE_CONCURRENCY`, default 3) per pass; the remainder is DEFERRED loud (a `decision.deferred` event + the returned `deferred` list) and re-offered next pass ŌĆö NEVER silently truncated.
